# Infraestructura Informática Empresarial

Proyecto integral de implementación de infraestructura IT para empresa de servicios con 50 empleados, incluyendo servidores, red corporativa y sistemas de backup.

## Características del Proyecto

### Alcance del Proyecto
- **Empresa**: Servicios profesionales, 50 empleados
- **Ubicación**: Oficinas centrales en Burgos
- **Tipo**: Implementación completa de infraestructura IT
- **Duración**: 3 semanas

### Sistemas Implementados

#### Servidores y Virtualización
- Servidor principal Dell PowerEdge R740
- Sistema de virtualización VMware vSphere
- 8 máquinas virtuales para diferentes servicios
- Sistema de alta disponibilidad y failover

#### Red Corporativa
- Switch principal Cisco Catalyst 48 puertos
- Switches secundarios por plantas
- Red WiFi empresarial con controlador centralizado
- Segmentación de red por departamentos

#### Sistemas de Backup
- Servidor de backup dedicado
- Sistema de backup automático diario
- Replicación offsite en la nube
- Pruebas de recuperación mensuales

#### Seguridad IT
- Firewall perimetral SonicWall
- Sistema antivirus corporativo
- Políticas de seguridad implementadas
- Monitorización 24/7 de la red

## Beneficios Obtenidos

- **Productividad**: 40% mejora en tiempos de respuesta
- **Seguridad**: Protección completa contra amenazas
- **Escalabilidad**: Sistema preparado para crecimiento
- **Disponibilidad**: 99.9% de uptime garantizado

## Servicios Incluidos

- Migración de datos sin interrupciones
- Formación del personal IT interno
- Documentación técnica completa
- Soporte técnico 24/7 primer año

## Tecnologías Utilizadas

- **Servidores**: Dell PowerEdge, VMware
- **Red**: Cisco, Ubiquiti, SonicWall
- **Backup**: Veeam, Azure Backup
- **Monitorización**: PRTG, Nagios
