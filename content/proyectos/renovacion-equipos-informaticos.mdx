# Renovación de Equipos Informáticos

Renovación completa del parque informático de una oficina con 25 estaciones de trabajo, incluyendo migración de datos y configuración personalizada.

## Características del Proyecto

### Alcance del Proyecto
- **Cliente**: Despacho de abogados
- **Equipos**: 25 ordenadores de sobremesa
- **Ubicación**: Centro de Burgos
- **Duración**: 1 semana

### Equipos Instalados

#### Ordenadores de Sobremesa
- 25 equipos Dell OptiPlex 7090
- Procesador Intel Core i7 de 11ª generación
- 16GB RAM DDR4
- SSD de 512GB + HDD 1TB

#### Periféricos
- Monitores Dell de 24" Full HD
- Teclados y ratones ergonómicos
- Impresoras multifunción en red
- Sistemas de backup externos

### Servicios Realizados

#### Migración de Datos
- Backup completo de datos existentes
- Migración automática de perfiles de usuario
- Transferencia de configuraciones personalizadas
- Verificación de integridad de datos

#### Configuración de Software
- Instalación de Windows 11 Pro
- Suite ofimática Microsoft 365
- Software específico del sector legal
- Configuración de seguridad corporativa

#### Integración en Red
- Configuración de dominio Active Directory
- Políticas de grupo aplicadas
- Acceso a recursos compartidos
- Configuración de impresoras en red

## Beneficios Obtenidos

- **Rendimiento**: 300% mejora en velocidad de procesamiento
- **Productividad**: Reducción de tiempos de espera
- **Seguridad**: Sistemas actualizados y protegidos
- **Fiabilidad**: Equipos con garantía completa

## Proceso de Implementación

1. **Análisis previo**: Inventario de equipos y software
2. **Planificación**: Cronograma de migración por fases
3. **Instalación**: Configuración de equipos nuevos
4. **Migración**: Transferencia de datos sin interrupciones
5. **Formación**: Capacitación del personal

## Garantías y Soporte

- **Hardware**: 3 años de garantía on-site
- **Migración**: Garantía de integridad de datos
- **Soporte**: 6 meses de soporte técnico incluido
- **Formación**: Sesiones de capacitación incluidas
