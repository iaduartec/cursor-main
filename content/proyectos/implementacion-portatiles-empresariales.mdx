# Implementación de Portátiles Empresariales

Proyecto de despliegue de 15 portátiles empresariales para equipo comercial con configuración remota y sistemas de gestión centralizada.

## Características del Proyecto

### Alcance del Proyecto
- **Cliente**: Empresa de consultoría
- **Equipos**: 15 portátiles empresariales
- **Modalidad**: Trabajo híbrido y remoto
- **Duración**: 5 días

### Equipos Suministrados

#### Portátiles Lenovo ThinkPad
- Procesador Intel Core i7 de 12ª generación
- 16GB RAM DDR5
- SSD NVMe de 1TB
- Pantalla 14" Full HD IPS

#### Accesorios Incluidos
- Docking stations para oficina
- Maletines de transporte
- Ratones inalámbricos
- Auriculares con micrófono

### Configuración Implementada

#### Gestión Centralizada
- Microsoft Intune para gestión de dispositivos
- Políticas de seguridad corporativas
- Actualizaciones automáticas centralizadas
- Control de aplicaciones instaladas

#### Seguridad Avanzada
- Cifrado completo de disco BitLocker
- Windows Hello para autenticación biométrica
- VPN corporativa preconfigurada
- Antivirus empresarial con gestión central

#### Software Corporativo
- Microsoft 365 con sincronización offline
- Software específico de consultoría
- Herramientas de colaboración Teams
- Aplicaciones de productividad

## Beneficios Obtenidos

- **Movilidad**: Trabajo desde cualquier ubicación
- **Seguridad**: Protección de datos corporativos
- **Productividad**: Herramientas optimizadas para trabajo remoto
- **Gestión**: Control centralizado de todos los dispositivos

## Servicios Incluidos

- Configuración remota de equipos
- Migración de perfiles de usuario
- Formación en herramientas de trabajo remoto
- Soporte técnico remoto 24/7

## Garantías

- **Hardware**: 3 años de garantía internacional
- **Software**: Licencias corporativas incluidas
- **Soporte**: 1 año de soporte técnico premium
- **Reemplazo**: Servicio de sustitución en 24h
